## 14. 제출 순서 요약

1. **나의 앱 → 새 앱 생성**
   - 플랫폼: iOS
   - 앱 이름, Bundle ID, SKU, 기본 언어 입력

2. **앱 정보 탭**
   - 카테고리 선택
   - 콘텐츠 권한 설정

3. **가격 및 사용 가능 여부**
   - 무료 선택
   - 모든 국가 선택

4. **앱 개인정보 보호**
   - 개인정보 처리방침 URL 입력
   - 데이터 수집 항목 설정

5. **버전 정보 (iOS 앱 1.0)**
   - 스크린샷 업로드
   - 프로모션 텍스트, 설명, 키워드 입력
   - 지원 URL 입력
   - 빌드 선택
   - 연령 등급 설정
   - 심사 정보 입력

6. **심사를 위해 제출** 클릭
