# 프로젝트 요구사항 정의 (PRD)

## 프로젝트 개요

### 프로젝트명
대만맛집정보 (Taiwan Food Guide)

### 목적
대만 여행자를 위한 맛집, 야시장, 관광지 정보를 제공하는 모바일 웹 애플리케이션

### 타겟 사용자
- 대만 여행을 계획 중인 한국인
- 대만 현지에서 맛집을 찾는 여행자
- 대만 음식에 관심있는 사용자

## 핵심 기능 요구사항

### 1. 홈 화면
- [ ] 검색 기능 (식당명, 음식, 야시장, 지역)
- [ ] 실시간 자동완성 검색 제안
- [ ] 퀵 카테고리 버튼 (가로 스크롤)
- [ ] 인기 맛집 섹션 (카테고리별 최고 평점)
- [ ] 야시장별 맛집 섹션 (필터 탭)

### 2. 카테고리 분류
- [ ] 면류 (우육면, 면선 등)
- [ ] 밥류 (루로우판, 지로우판 등)
- [ ] 만두 (샤오롱바오 등)
- [ ] 우육탕
- [ ] 훠궈
- [ ] 디저트 (빙수, 두부 디저트 등)
- [ ] 길거리음식 (지파이, 굴전 등)
- [ ] 카페 (스페셜티 커피, 전통 찻집)
- [ ] 공차 (버블티)
- [ ] 까르푸 (대형마트)

### 3. 야시장 정보
- [ ] 스린 야시장
- [ ] 닝샤 야시장
- [ ] 라오허제 야시장
- [ ] 통화 야시장
- [ ] 펑자 야시장 (타이중)
- [ ] 단수이 야시장

### 4. 도심투어
- [ ] 시먼딩
- [ ] 융캉제
- [ ] 중산 카페거리
- [ ] 신이 지구

### 5. 갈만한 곳
- [ ] 관광 명소 정보
- [ ] 위치 및 특징 안내

### 6. 맛집 상세 정보
- [ ] 기본 정보 (이름, 위치, 특징)
- [ ] Google Places 연동 이미지
- [ ] 평점 표시
- [ ] 추천 메뉴
- [ ] 외부 링크 (Google Maps, 검색)

### 7. 리뷰 시스템
- [ ] 회원가입/로그인 (이메일 기반)
- [ ] 리뷰 작성 (별점 + 텍스트 + 사진)
- [ ] 세부 별점 (음식/서비스/분위기)
- [ ] 사진 업로드 (최대 4장, 자동 리사이즈)
- [ ] 본인 리뷰 삭제

### 8. PWA 지원
- [ ] 홈 화면 추가 기능
- [ ] 모바일 최적화 UI
- [ ] iOS Safe Area 지원

## 비기능 요구사항

### 성능
- 이미지 업로드 제한: 800px, JPEG 60%
- API 응답 시간: 2초 이내
- 초기 로딩: 3초 이내

### 보안
- JWT 토큰 인증 (httpOnly 쿠키)
- 비밀번호 bcrypt 암호화
- API 인증 필수 (리뷰 작성/삭제)

### 호환성
- 모바일 브라우저 (Chrome, Safari)
- iOS/Android 웹뷰
- 반응형 디자인 (최대 448px)

## 기술 스택

### 프론트엔드
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS 4
- Radix UI (shadcn/ui)
- Lucide Icons

### 백엔드
- Next.js API Routes
- MongoDB Atlas
- Cloudinary (이미지)
- JWT 인증

### 인프라
- Vercel (배포)
- GitHub (소스 관리)
